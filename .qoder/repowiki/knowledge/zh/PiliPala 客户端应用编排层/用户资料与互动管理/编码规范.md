- UseCase 类均包含一个 `execute` 方法作为唯一执行入口，并在失败时抛出包含错误信息的 Exception。
- Repository 方法统一返回泛型 `ApiResponse<T>` 对象，由上层 UseCase 或 Controller 判断 `isSuccess` 并提取数据。
- Controller 中的异步加载方法均采用“设置 loading -> try/catch 执行用例 -> finally 取消 loading”的标准状态管理模式。
- UI 组件采用原子化拆分，将个人页的不同板块（如 Profile, Coins, Likes）封装为独立的 StatelessWidget 并置于 `widgets/` 目录下。
