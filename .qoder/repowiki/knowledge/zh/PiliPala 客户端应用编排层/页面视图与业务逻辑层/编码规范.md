- 控制器类继承 `GetxController`，并在 `onInit` 中初始化状态、读取本地配置或发起初始网络请求。
- 视图文件通常定义一个 `StatefulWidget` 或 `StatelessWidget`，并通过 `Get.put` 或 `Get.find` 获取对应的 Controller 实例进行数据绑定。
- 本地存储操作封装在 `GStrorage` 工具类中，控制器通过访问特定的 Box（如 `GStrorage.setting`）来读写键值对。
- 异步网络请求的结果通常包装为 `{status: bool, data: any, msg: string}` 格式，控制器根据 `status` 更新 UI 状态或弹出提示。
- 设置类页面使用统一的 `SetSwitchItem` 或 `SelectDialog` 组件来处理布尔开关和枚举选择，并直接同步到 Hive 存储。
