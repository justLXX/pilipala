- **模块化分层**：按业务域划分子目录（如 `video/`, `live/`, `danmaku/`, `user/`），每个子目录包含对应的响应模型或配置枚举。
- **协议支持**：在 `danmaku/` 中通过 `.proto` 文件定义弹幕服务接口，并生成 Dart 绑定代码（`.pb.dart`）以支持高性能二进制数据传输。
- **本地存储集成**：核心用户模型（如 `UserInfoData`）使用 `Hive` 注解（`@HiveType`, `@HiveField`）并配合代码生成（`.g.dart`）实现高效的本地 NoSQL 持久化。
- **统一解析规范**：所有 API 响应模型均遵循 `fromJson` 构造函数模式，负责将后端 JSON 数据映射为强类型 Dart 对象，部分模型包含数据清洗逻辑（如 HTML 标签移除）。
