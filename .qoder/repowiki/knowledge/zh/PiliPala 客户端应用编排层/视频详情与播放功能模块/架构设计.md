- 采用 Clean Architecture 分层结构：`data` 层定义 `VideoDetailRepository` 封装网络请求；`domain` 层定义多个 UseCase（如 `GetVideoDetailUseCase`）处理业务逻辑；`presentation` 层包含 `VideoDetailPage`（UI）、`VideoDetailController`（状态管理）及自定义 Widget。
- 使用 GetX 进行依赖注入和响应式状态管理，`VideoDetailController` 协调播放器生命周期与数据加载。
- 通过 `video.dart`  barrel file 统一导出各层核心类，简化外部引用。
