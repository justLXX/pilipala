- 采用 Clean Architecture 分层设计：`data/` 层封装 `LoginRepository` 处理 API 交互与 RSA 加密；`domain/` 层定义细粒度 Use Cases（如 `LoginByPasswordUseCase`）；`presentation/` 层通过 `LoginController` (Getx) 协调 UI 状态与业务逻辑。
- 依赖注入基于 `get_it`/`Get.find` 模式，Repository 与 Use Cases 在构造函数中按需解析。
- 模块入口 `login.dart` 统一导出各层核心组件，确保外部引用的简洁性。
