- Repository 方法统一返回 ApiResponse<T>，并在 UseCase 中校验 isSuccess 后转换为领域模型或抛出异常。
- Controller 暴露私有 Rx 状态与公开 getter，页面通过 Obx 订阅变化并驱动 UI 渲染。
- 列表页采用 FutureBuilder 触发首次加载，完成后切换为 Obx 渲染 SliverList，错误时展示 HttpError 并提供重试。
- 滚动触底分页通过 ScrollController 监听距底部阈值触发 loadMore/loadMoreHot，并以 isLoadingMore 标志防重入。
- 数据过滤在 Repository 层执行，基于 goto 类型与本地黑名单 mid 列表进行剔除后再返回。
