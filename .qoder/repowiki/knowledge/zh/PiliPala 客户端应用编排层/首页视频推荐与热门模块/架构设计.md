- 采用 Clean Architecture 分层：data（VideoRepository）、domain（UseCases）、presentation（Controller + Pages）。
- 入口为 home.dart 的 barrel exports，统一暴露各层核心类。
- 依赖方向：presentation → domain → data → 核心网络 ApiClient；通过 GetX 进行依赖查找与状态管理。
- 表现层由 HomePage 组织 TabBar/TabBarView，分别承载 RcmdPage 与 HotPage；HomeController 维护推荐与热门两套独立状态与加载流程。
