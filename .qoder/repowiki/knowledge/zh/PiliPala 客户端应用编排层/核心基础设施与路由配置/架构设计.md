- 分层结构：`lib/core` 定义抽象接口（`ApiClient`, `StorageService`）及单例服务（`ThemeService`）；`lib/http` 实现底层 Dio 客户端初始化、拦截器链及业务 API 分类；`lib/router` 基于 GetX 管理页面路由与依赖绑定；`lib/services` 处理后台音频等系统级服务。
- 依赖方向：高层模块通过 `DependencyInjection` 和 `Bindings` 组装底层服务，遵循控制反转原则。
- 关键文件：`lib/core/di/dependency_injection.dart` 负责全局服务注册；`lib/http/init.dart` 配置 Dio 实例；`lib/router/app_pages.dart` 定义路由表。
